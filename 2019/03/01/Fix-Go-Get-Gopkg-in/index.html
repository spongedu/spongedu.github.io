go get gopkg.in报错及修复

发表于 2019-03-01

最近不小心删掉了go mod的Cache, 在重新编译TiDB的时候发现无法下载一些依赖，报错go get Unknown SSL protocol error in connection to gopkg.in。
Google了一下，发现github上有人反馈一样的问题Unknown SSL protocol error in connection to gopkg.in:-9838。
具体解决办法如这篇博客所述, 需要修改git配置如下: git config --global http.sslVerify true
